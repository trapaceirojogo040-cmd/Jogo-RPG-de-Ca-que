# Jogo-RPG-de-Caíque: Estratégia Militar Avançada

Simulação estratégica profunda. Gerencie a Base Militar, recursos (Ouro, Éter) e tecnologia ramificada. O Protocolo de IA oferece análises de risco. Recrute Personagens com habilidades detalhadas e Afinidade (Harém/Romance) para aprimorar a Força Bélica. Seinen/Fantasia/Drama.
