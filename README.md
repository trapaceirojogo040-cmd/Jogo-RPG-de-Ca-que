# Jogo-RPG-de-Ca-que
Jogo RPG de Caíque: Simulação de Estratégia Militar Avançada. Gerencie sua Base Militar, recursos (Ouro, Éter) e árvore tecnológica. O Protocolo de IA oferece análise de risco. Recrute Personagens com habilidades detalhadas e sistema de Afinidade (Harém/Romance) para aumentar sua Força de Combate. Gêneros: Seinen/Fantasia/Drama.
