# Jogo RPG de Caíque: Estratégia Militar Avançada

Simulação estratégica profunda. Gerencie sua Base Militar, recursos (Ouro, Éter) e árvores de tecnologia. O Protocolo de IA oferece análises de risco. Recrute personagens com habilidades detalhadas e Afinidade para aprimorar a Força Bélica. Gêneros: Seinen/Fantasia/Drama.
