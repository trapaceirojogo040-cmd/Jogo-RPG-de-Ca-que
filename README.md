# Jogo-RPG-de-Caíque
Jogo RPG de Caíque: Estratégia Militar Avançada. Uma simulação estratégica profunda. Gerencie a Base Militar, recursos (Ouro, Éter) e uma tecnologia ramificada. O Protocolo de IA oferece análises de risco. Recrute Personagens com habilidades detalhadas e Afinidade (Harém/Romance) para aprimorar a Força Bélica. Gêneros: Seinen/Fantasia/Drama.
